# FREERTOS-mushroom-monitoring
Mushroom monitoring using real time operating system FreeRTOS

## Component
- Arduino UNO
- DHT 22
- LDR
- LCD 16x4

## Task
1. LCD Task (`lcd_task`)
2. DHT22 Task (`dht22_task`)
4. LDR task (`ldr_task`)
